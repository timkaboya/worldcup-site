# World Cup 2026 Companion

A fast, lightweight web app for following the **FIFA World Cup 2026** in real time — live
scores, match stats, group tables, top scorers, the knockout bracket, and curated news —
localized to **your timezone**, on mobile and web, with **no sign-up**.

Built for the duration of the tournament: static-first, installable (PWA), and backed only by
a thin read-only edge data layer. No accounts, no user data.

## Status

Pre-implementation. We are finalizing specs before building.

## Principles

- Lightweight first · No accounts · Your timezone · Mobile + web · Authentic, attributed content.
